Ghost on Github Page: 這個網站是怎麼生出來的

有玩 Github 的讀者都知道，它佛心地為使用者提供了一套免費的靜態網頁託管服務: Github Pages。為很多想要自己管理一套靜態網頁（如：個人網站、部落格、開源社群或產品官網等網頁）的使用者提供了極大的便利。不然的話，各家雲服務提供商的託管及域名管理等費用，每個月算下來也是一筆不痛但又很癢的固定開銷。

筆者「入行」多年，前前後後在各平台，使用不同的後台系統，架設過多次網站。拿部落格系統來說，Github 官方推薦使用的 Jekyll、神通廣大的 Wordpress 以及短小精悍的 Ghost Blog 都有所涉略。但筆者還是最喜歡 Ghost Blog，它開源免費、輕量便捷、能夠快速集成插件、美化前端簡單、支援網頁自適應等優點，讓它成為了架設個人部落格的首選系統。這此提供了一些 showcase 供讀者參考。

【題外話】近年來很多年輕人都開始使用 Hexo，創作者是台灣人 Tommy351，其官網也提供了架設至 Github Pages 的方法，這裡列出來供各位參考：https://hexo.io/zh-tw/docs/github-pages.html

然而，Ghost Blog 的背後是一個 Node.js 後台系統，為了能夠將它架設至僅支援靜態頁面的 Github Pages 上，要如何進行呢？沒錯！聰明的讀者一定想到了：只需要在本地端架設部落格系統，然後將所有前端靜態資源抓出來，往 Github Pages 上放即可！

接下來，筆者將會就此部分展開介紹。

1] 在本地端架設 Ghost Blog

只需要遵循上述官方文件中的步驟進行安裝即可，由於官方網站描述十分清晰，這裡就不重複介紹。推薦使用 Docker 的版本，因為本地安裝需要使用者的環境具備指定版本的 Node。

安裝好之後，在默認情況下，Ghost Blog 將會在 https://davisxching.github.io 被訪問到，而通過 https://davisxching.github.io/ghost/  則可以對您的部落格進行管理。

2] Github Pages 如何使用

要在 Github Pages 上架設網頁則更加簡單，只需要建立一個名為 your-username.github.io 的 Repository 即可！（這裡的 your-username，指的是您 Github 用戶名。）

在建立完成之後，訪問 https://your-username.github.io 就會默認導向這個 repo 中的 index.html。在每次上版之後，一般需要等待一段時間 (通常是幾秒至一分鍾不等)，網頁才會產生變化。

3] Ghost on Github Pages

那麼接下來，只需要將本地架設的 Ghost Blog 前端資源扒出來，再 push 至我們的 Github Pages 的 Repository 即可。

要做到這一點，目前主要有以下兩個比較熱門的專案可以參考。

第一個是國外的一名工程師 Paladini 的專案 。他運用了一隻很古早的 npm 套件(buster，[Ghost buster?])，外加上自己的腳本，實現了這個過程。但是筆者目前並不推薦使用這個專案，原因是因為這個 buster 套件已經年久失修，且僅支援低版本的 Node。其二是因為這位工程師的腳本也有些許不足之處，導致還得手動去優化一些內容，才能讓整個過程走的更順。

而另一隻比較推薦的專案是：ghost-static-site-generator。此方案安裝簡單，使用也方便。只需要一個指令就可以將本地的 Ghost Blog 靜態頁面產出至指定地點，爾後直接推送至上文中提到的 Github Pages' Repositoy 即可。

4] 美化網頁

遵循上文中的三大步驟，網頁已經可以成功的被訪問了。但為了讓您的網頁更加吸睛，則勢必需要進行一些美化工作。要在 Ghost Blog 中進行美化是非常方便的，大部分資料都可以在 官方介紹 中找到，如添加主題、修改 CSS 或者更改 HTML 等。筆者就不對這些提到的內容進行描述，在此僅介紹官方文檔中未提到的內容：快速修改字體 與 程式碼美化。

快速修改字體

最快速、便捷的方式就是採用 Google Fonts。可以將字體下載下來，打包至主題裡面。當然，更方便的作法就是遵循它提供的 html 程式碼，直接添加至 header 中即可。爾後只需要將欲使用該字體的元素加入 CSS 即可。這裡提供此網站中使用的方法供大家參考：

<link href="https://fonts.googleapis.com/css2?family=Lobster&family=Noto+Serif+TC:wght@200;300;400;500;600;700;900&display=swap" rel="stylesheet">
<style>
    
    .title, #header nav ul.menu li a, div.text, div.text h1, div.text h2, div.text h3, div.text h4, div.text h5, div.text h6, a, div.tags a, .widget-tag-cloud a, div.articles article div.post-excerpt {
    	font-family: 'Noto Serif TC', serif;
    }
    
    a.title.large.white.no-margin, a.title.white.no-margin {
        font-size: 3em;
        font-family: 'Lobster', cursive;
    }
    
</style>

程式碼美化

工程師的部落格少不了對程式碼進行展示、分享，這裡就會需要對程式碼進行美化與高亮(因程式語言不同而異)，即對  <pre> 或 <code> 元素進行 CSS 處理。這裡就找到了完美的解方：highlight.js 。

以下是在此網站中使用 highlight.js 進行美化的方法：

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.5.0/styles/rainbow.min.css">
<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.5.0/highlight.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.5.0/languages/java.min.js"></script>

<script type="text/javascript">
    window.onload = function() {
  		hljs.highlightAll();
    }
</script>

結語

本篇文章快速地介紹了此網站的架設方法，以及所有參考連結，希望可以幫到也想將酷酷的 Ghost Blog 架設至 Github Pages 的您。
